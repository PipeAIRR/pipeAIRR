3  How to run a pipeline in dolphinNext?

1. Launch your dolphinnext docker

2. Navigate to http://localhost:8080/dolphinnext/

3. Choose your pipeline

4. Click on “Run” button

5. Click on “Create New Run” option

6. Choose the Project or create a new one

7. Click on “Select Project” button

8. Enter run name and click on “Save run”

9. Choose run enviroment

10. Enter work directory

11. Enter the files and values the pipeline need

12. Click on “Run” button

13. Choose “Start” option

14. For more information how to run pipeline go to

https://dolphinnext.readthedocs.io/en/latest/dolphinNext/project.html

15. For information about the run page go to https://dolphinnext.readthedocs.io/en/latest/dolphinNext/run.html
